---
title: "Keyboard Shortcuts"
description: "Keyboard shortcuts supported on the Emergent app. Uses Mod to represent the platform modifier key (⌘ on macOS, Ctrl elsewhere)."
---

Emergent does not override standard browser shortcuts. The list below covers
the bindings that interact with the app surface itself.

## Global

| Shortcut         | Action                                                      |
|------------------|-------------------------------------------------------------|
| **Escape**       | Close the foreground modal (Pricing, Referral, Settings).   |
| **Mod+K**        | Focus the global search (where enabled by context).         |
| **Mod+/**        | Toggle the keyboard shortcuts help overlay (where enabled). |

## Task input

| Shortcut         | Action                                                      |
|------------------|-------------------------------------------------------------|
| **Enter**        | Insert a newline in the prompt textarea.                    |
| **Mod+Enter**    | Submit the task (equivalent to pressing the submit arrow).  |
| **Shift+Tab**    | Move focus backward through the build-mode tabs.            |
| **Tab**          | Move focus forward through the build-mode tabs and controls.|

## App builder

| Shortcut         | Action                                                      |
|------------------|-------------------------------------------------------------|
| **Mod+W**        | Close the active build tab (matches browser behavior).      |
| **Mod+Shift+T**  | Reopen the most recently closed build tab.                  |
| **Mod+1 … Mod+9**| Switch directly to tab 1 through 9.                         |

## Language picker

| Shortcut         | Action                                                      |
|------------------|-------------------------------------------------------------|
| **Arrow keys**   | Navigate the locale list.                                   |
| **Enter**        | Apply the highlighted locale.                               |
| **Type to search** | Filter the locale list using the inline search input.     |

## Preview tab

| Shortcut         | Action                                                      |
|------------------|-------------------------------------------------------------|
| **Mod+R**        | Refresh the previewed application (iframe reload).          |
| **Mod+Shift+O** | Open the preview in a new browser tab.                       |

Tool-specific shortcuts inside a generated app (e.g. a remixed IDE or dashboard)
belong to that template — Emergent does not intercept them.
