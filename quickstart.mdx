# Build Your First App

Follow these steps to go from idea to deployed app using Emergent.

## Step 1: Sign Up

Go to [app.emergent.sh](https://app.emergent.sh) and create an account using Google, GitHub, Apple, Facebook, email, or phone number. New accounts receive **10 free credits** to get started.

## Step 2: Choose Your App Type

On the home dashboard, select the type of application you want to build:

<CardGroup cols={3}>
  <Card title="Full Stack App" icon="layer-group">
    Complete web applications with frontend and backend.
  </Card>
  <Card title="Mobile App" icon="mobile">
    Mobile-first applications.
  </Card>
  <Card title="Landing Page" icon="browser">
    Marketing and product landing pages.
  </Card>
</CardGroup>

## Step 3: Describe Your App

Type a description of what you want to build in the prompt box. Be as specific or general as you like.

<img
  src="/images/home-dashboard.png"
  alt="Emergent prompt input"
/>

**Example prompts:**
- "Build me a dashboard for tracking fitness goals"
- "Create a landing page for my SaaS product with pricing and FAQ sections"
- "Build a mobile app for tracking daily expenses"

## Step 4: Select an AI Model

Click the model selector to choose which AI model powers your build. **Claude 4.5 Sonnet** is selected by default and works well for most use cases.

<img
  src="/images/model-selector.png"
  alt="Model selector dropdown"
/>

## Step 5: Set Visibility

Choose whether your app is **Public** (visible in the Showcase) or **Private**.

## Step 6: Build

Click the send button to start building. The AI agent will:

1. Analyze your requirements
2. Generate the code
3. Set up the frontend and backend
4. Show you a live preview

<img
  src="/images/app-builder-complete.png"
  alt="App builder with live preview"
/>

## Step 7: Iterate

Once the initial build is complete, use the chat to request changes:
- "Change the background color to dark blue"
- "Add a contact form at the bottom"
- "Make the navigation sticky"

The agent will update your app in real time.

## Step 8: Deploy

When you're happy with your app, click the **Deploy** button in the preview panel to make it live.

<Note>
  Each build consumes credits based on the AI model used and the complexity of the task. Check the [Credits & Billing](/guides/credits-and-billing) guide for details.
</Note>
