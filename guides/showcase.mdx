# Showcase

The Showcase is a gallery of apps built by the Emergent community. Browse for inspiration, preview live apps, and remix them into your own projects.

<img
  src="/images/home-dashboard.png"
  alt="Showcase section on home page"
/>

## Browsing the Showcase

The Showcase section appears on the home dashboard below the app builder prompt. Filter apps by category:

| Category | Description |
|----------|-------------|
| **All** | View all showcase apps |
| **Landing Pages** | Marketing pages, product showcases, promotional sites |
| **Advanced Apps** | Complex full-stack applications |
| **Business Tools** | Tools for business operations and productivity |
| **Personal Tools** | Apps for personal use and productivity |
| **Mobile Apps** | Mobile-optimized applications |

## App Cards

Each showcase app is displayed as a card with:

- **Preview thumbnail** — A screenshot of the app
- **App name** — The title of the application
- **Category** — Which showcase category it belongs to

### Actions

Hover over an app card to reveal action buttons:

- **Preview** — Opens the app in a read-only view to explore the full application
- **Remix** — Creates a copy of the app in your project so you can customize it

## Previewing Apps

Click **Preview** to open an app in the preview interface:

<img
  src="/images/app-preview.png"
  alt="App preview interface"
/>

The preview interface shows:
- **App description** on the left — a summary of what the app does
- **Live app** on the right — the fully functional app running in an iframe
- **Device toggle** — Switch between desktop, tablet, and mobile views
- **Remix this template** — Create your own editable copy

## Remixing Apps

Click **Remix** or **Remix this template** to create your own copy of a showcase app. This opens the full app builder where you can:

1. Chat with the AI agent to request modifications
2. Change the design, add features, or remove elements
3. Deploy your customized version

<Tip>
  Remixing is a great way to learn — start with a polished app and modify it to understand how the builder works.
</Tip>
