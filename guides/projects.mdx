## Overview

A **Project** is the top-level container for everything you build on Emergent.
Tasks, preview tabs, credit spend, and member access all scope to a project.
Every account ships with one default project (named after your profile) and
can create additional projects from the workspace switcher above the Home
hero.

## Membership

- Each project has exactly one **Owner** (the account that created it).
- Projects show a "• N member(s)" hint next to the name. Inviting collaborators
  moves the project from a personal to a shared workspace.
- Access-controlled features (private builds, shared GitHub repos) scope to
  the project; changing the active project hot-swaps the task input's project
  pill accordingly.
