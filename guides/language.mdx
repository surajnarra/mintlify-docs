## Overview

The **Language** shortcut in the avatar menu opens a searchable locale picker.
Selecting a language reloads the app with translated UI strings. The same
setting is also available from [Preferences](/guides/preferences); changes in
either location stay in sync.

## Picker behavior

The picker includes an inline search input — type a partial name (for example
`port`) to narrow the list. Keyboard navigation with arrow keys plus `Enter`
activates a locale without the mouse.
